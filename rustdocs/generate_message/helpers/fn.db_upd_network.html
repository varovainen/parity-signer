Update the database after add_specs run.

Inputs &str url address that was used for rpc calls and already completed NetworkSpecsToSend.

Adds simultaneously AddressBookEntry to ADDRESS_BOOK and NetworkSpecsToSend to SPECSTREEPREP.

Key for AddressBookEntry is the network address book title. It always has format <network_name>-<network_encryption>.
