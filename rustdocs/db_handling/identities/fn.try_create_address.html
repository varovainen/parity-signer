Create a new address in the Signer database.

If the associated AddressKey already exists in the ADDRTREE tree of the Signer database, a new network NetworkSpecsKey is added into network_id set of the AddressDetails. If the AddressKey is new, a new ADDRTREE entry is made.

Secrets passed into this function are:

  • seed phrase
  • path, possibly with secret password part (///<...>).

Both are input as &str and go directly in create_address (both) and derivation_check (path only).
